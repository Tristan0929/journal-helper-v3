📖 使用教程

一个把外刊文章变成个人学习笔记、并练读写听说的小工具。所有数据只存在你自己的浏览器里。

① 准备

点右上角 ⚙️ 设置，填入你自己的 API Key（默认 DeepSeek，也可选 OpenAI / Kimi / 智谱 等）。原文语种默认自动识别，日/德/韩/西等外刊都行，识别不准可手动指定。密钥只存在你浏览器，服务器只转发不保存。

② 生成笔记

把外刊文章直接粘进输入框——分栏、断行、乱码都不用管，AI 会自动理顺版面。点🚀 开始分析，笔记会像打字一样流式出现；中途发现跑偏，点⏹ 停止重生成可立刻中断、省 token。笔记含六块：文章 / 词汇精析 / 长难句语法 / 句子赏析 / 写作手法 / 背诵速查表。

红色加粗=重点词汇   蓝色斜体=精彩句子   灰色加粗=结构标签（导出 PDF 颜色保留）。

③ 上方标签切换功能

🎴 闪卡：翻卡背词，翻面后用 不认识 / 模糊 / 认识 记熟练度（简易 Anki），可「只复习不熟的」，还有 ✍️ 拼写自测，也能导出 Anki CSV。
📇 词汇表：单独看速查表，导出 PDF / CSV。
✍️ 练习题：可选两栏（左文右题，选中文字可黄色划重点，不导 PDF）或上下（文章在上题目在下，可把文章＋题目一起导 PDF）；点「生成题目」才出 5 道阅读选择题，即时判分。
🌍 深度思考：点「生成」出一段批判性分析，默认用原文的语言、也可下拉自选输出语言；下方可用该语言写评论，AI 先润色你的表达再和你讨论（练写作）。可 🔊 朗读、🌐 译成中文（点了才翻、可隐藏）；分析＋讨论一起导出 PDF 或复制 MD。

④ 选中：问 AI / 添加单词

在笔记或深度思考里选中词句，旁边冒出 💬 问 AI（带上下文提问）和 ＋添加单词（把该词加进词汇表/闪卡；在设置里可选自动补释义或手动填）。

⑤ 保存与清空

文章、笔记、问答、深度思考讨论、闪卡熟练度都会自动存在本地浏览器，刷新/关掉再打开都还在，直到你点 🗑️ 清空重来 开始下一篇。

⑥ 朗读（听力）

笔记和深度思考页有 🔊 朗读：点了会自动复制排版好的纯文本并打开 NaturalReaders，在网页里粘贴（Ctrl/⌘+V）即可用它更自然的真人音色朗读、调语速。

⑦ 导出

Markdown 一律「复制」（省得下载）；PDF 各板块单独下载。打印对话框记得勾「背景图形」以保留配色。

⑧ 字体 & 装成 App

设置里可调字号和字体。想像 App 一样用：手机浏览器打开后「分享 → 添加到主屏幕」，就有独立图标、全屏打开（PWA，免费、无需上架）。右下角 💬 按钮可拖动换位置。

添加单词

手动填写释义和场景，加入词汇表和闪卡。
