# 外刊阅读助手 v3

一个把外刊英文文章做成个人学习笔记的小工具。相比 v2 的改动：

- **流式输出**：边生成边显示，不用再干等一整份笔记，体感快很多。
- **自动整理版面**：直接粘贴分栏、断行、乱码的原文，AI 会在「文章展示」板块先把版面理顺，再做笔记，不用手动排版。
- **真正可配置**：右上角「设置 API」可选 DeepSeek（默认）/ OpenAI / Kimi / 智谱 / 自定义，服务商、base_url、模型、Key 全部前端可配，后端动态读取。
- **不预置任何 Key**：每个用户填自己的 Key，只存在各自浏览器里，服务器只转发、不存储。
- **两个独立 PDF 导出**：笔记一份、对话一份，互不干扰。
- **自动保存**：最近一次的文章、笔记、对话存在浏览器，刷新不丢。
- **侧边栏选中提问**：保留，且可自由提问；AI 先常规解释，文中用法有独特处再补充。
- **防冷启动**：新增 `/ping` 端点，可用定时任务每隔几分钟访问一次，避免免费主机休眠导致首次很慢。

## 本地运行

```bash
pip install -r requirements.txt
python app.py
# 打开 http://localhost:8080
```

## 部署（生产）

```bash
gunicorn app:app --bind 0.0.0.0:$PORT --workers 1 --threads 8 --timeout 300
```

> 流式输出是长连接，`--timeout` 要设大一点（如 300 秒）；`--threads` 让单 worker 也能并发，适合 512MB 内存的小机器。

## 防冷启动

免费主机在没人访问时会休眠，导致下次打开要先花几十秒唤醒。部署后可用任意定时服务（如 cron-job.org 或 UptimeRobot）每 5 分钟请求一次：

```
https://你的域名/ping
```

保持服务器常醒，别人打开就是秒开。
