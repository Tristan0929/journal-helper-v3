# 外刊阅读助手 v3

> Copyright © 2026 **Tj** · [github.com/Tristan0929](https://github.com/Tristan0929) · MIT License

把外刊文章变成个人学习笔记，并练读、写、听、说的小工具。粘贴文章即可，所有数据只存在用户自己的浏览器里，服务器只做转发、不存储任何内容或密钥。支持任意外语（英/日/德/韩/西等），可装成手机 App（PWA）。

> 开源、MIT 许可。欢迎自由使用与二次开发，但请**保留版权署名（Tj）**——见 `LICENSE`。

## 功能一览

### 笔记（六板块，流式生成）
- **自动整理版面**：直接粘贴分栏、断行、乱码的原文，AI 先把版面理顺再做笔记，无需手动排版。
- 六个板块：文章展示 → 词汇词组精析（脚注）→ 长难句语法精讲 → 句子赏析 → 写作手法 → 背诵速查表。
- **词汇宽收、句子精选**：词汇尽量多收地道搭配/习语/进阶词；精彩句子和结构精妙的长难句在文章里高亮。
- 高亮规则：<span style="color:#c0392b">红色加粗</span>=重点词汇、*蓝色斜体*=精彩/结构好的句子（**斜体只出现在原文**，赏析/语法/写作手法里引用一律纯文本）、灰色加粗=结构标签。导出 PDF 颜色保留。

### 标签页功能
- **🎴 闪卡（简易 Anki）**：翻面后用「不认识 / 模糊 / 认识」记熟练度、可「只复习不熟的」；另有 ✍️ 拼写自测；熟练度存本地；导出 Anki CSV。
- **📇 词汇表**：单独查看背诵速查表，导出 PDF / CSV。
- **✍️ 练习题**：生成前可选**两栏**（左文右题，选中文字可黄色划线高亮、存本地，不导 PDF）或**上下**（文章在上题目在下，可把文章＋题目一起导出 PDF）；文章为**纯黑无标注**版；点「生成题目」才出 5 道选择题，即时判分。
- **🌍 深度思考**：点「生成」出一段批判性分析，默认**用原文的语言**、也可下拉自选输出语言；下方可用该语言写评论，AI **先润色你的表达再和你讨论**（练写作）；可 🔊 朗读、🌐 译成中文（点了才翻、可隐藏）；分析＋讨论一起导出 PDF 或复制 MD。

### 交互与体验
- **选中提问 / 添加单词**：在**笔记或深度思考**里选中词句 → 冒出「💬 问 AI」与「＋添加单词」；问 AI 带上下文；添加单词按设置里「自动补释义（调 AI）/ 手动填」并入词汇表与闪卡。手机上「选中→点右下角 💬」也会自动带上选中内容。
- **可拖动的 💬 悬浮按钮**：拖动换位、记住位置，轻点开关侧边栏。
- **停止重生成**：流式生成中可随时中断，省 token。
- **翻译**：深度思考的分析、讨论里 AI 的每条回复都有 🌐 译按钮，点了才翻成中文（消耗一次 API），结果缓存、可随时隐藏。
- **多语言**：默认自动识别原文语种；笔记讲解用中文，深度思考/讨论用原文语言；设置里可手动指定语种兜底。
- **字体**：设置里可调字号（小/标准/大/特大）和字体（系统/衬线/圆润/等宽）。
- **本地自动保存**：文章、笔记、问答、深度思考讨论、闪卡熟练度、练习题、翻译全部存浏览器，刷新/重开都在，直到点「🗑️ 清空重来」进入下一篇。
- **导出**：Markdown 一律「复制」，PDF 各板块单独下载。
- **📖 使用教程**：右上角按钮，随时查看详细教程。

### 朗读（听力）
笔记和深度思考页有 🔊 朗读：点了会自动复制排版好的纯文本并打开 [NaturalReaders](https://www.naturalreaders.com/online/)，粘贴即可用它的真人音色朗读、调语速。

### PWA（免费装成 App）
手机浏览器打开后「分享 → 添加到主屏幕」，即有独立图标、全屏打开，无需上架、不花钱。需 HTTPS（Render 默认满足）。正式上 App Store / TestFlight 则需 Apple 开发者账号（99 美元/年）+ Mac 打包，非免费。

### 服务与配置
- **多服务商可配置**：DeepSeek（默认）/ OpenAI / Kimi / 智谱 / 自定义，前端可配、后端动态读取。
- **不预置任何 Key**：每个用户填自己的 Key，只存各自浏览器。
- **防冷启动**：`/ping` 端点，配定时任务保活。

## 接口

| 路由 | 说明 |
|------|------|
| `POST /analyze` | 生成六板块笔记（流式 SSE） |
| `POST /chat` | 侧边栏 AI 导师（流式） |
| `POST /exercises` | 生成阅读理解题（JSON） |
| `POST /deepdive` | 深度思考，默认原文语种、可指定输出语言（流式） |
| `POST /discuss` | 深度思考的讨论 + 写作润色（流式） |
| `POST /translate` | 翻译成中文（流式） |
| `POST /defineword` | 给自定义词补释义/场景（JSON） |
| `GET /manifest.webmanifest` · `GET /sw.js` | PWA 清单与 service worker |
| `GET /ping` | 保活端点 |

> 建议在 `.gitignore` 里加上 `__pycache__/`，避免提交 Python 缓存。

## 本地运行

```bash
pip install -r requirements.txt
python app.py          # 打开 http://localhost:8080
```

## 部署（生产）

```bash
gunicorn app:app --bind 0.0.0.0:$PORT --workers 1 --threads 8 --timeout 300
```

流式是长连接，`--timeout` 要大；`--threads` 让单 worker 也能并发，适合 512MB 小机器。部署时记得**一并上传 `static/` 图标**，PWA 需 HTTPS。

## 防冷启动

用 cron-job.org / UptimeRobot 每 5 分钟访问一次 `https://你的域名/ping` 即可保持常醒。

## 小贴士

- 闪卡 / 词汇表依赖「背诵速查表」，等笔记整份生成完再切过去。
- 导出 PDF 时在打印对话框勾选「背景图形 / Background graphics」，配色和底纹才完整保留。
- 深度思考、讨论、翻译质量对模型较敏感，`deepseek-v4-pro` 比 `flash` 更稳。
