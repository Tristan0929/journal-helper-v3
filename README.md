# 外刊阅读助手 v3

把外刊英文文章做成个人学习笔记的小工具。

## 功能

- **笔记**：粘贴分栏/乱码原文，AI 自动理顺版面后生成五板块笔记（文章展示 / 词汇精析 / 句子赏析 / 写作手法 / 背诵速查表），流式输出边生成边看。
  - 配色克制：**红色**=重点词汇、*蓝色斜体*=精彩句子、灰色加粗=结构标签；导出 PDF 颜色保留。
- **闪卡**：用词汇表做成翻卡，可翻面 / 上一张 / 下一张 / 打乱，导出 Anki CSV。
- **词汇表**：抽取背诵速查表单独查看，导出 PDF 或 CSV。
- **练习题**：根据文章现出 5 道阅读理解选择题（英文题干 + 中文解析），点选项即时判分。
- **侧边栏 AI 导师**：自由提问；在笔记里选中词句会自动带入上下文，AI 先常规解释、文中用法有独特处再补充。
- **独立导出**：笔记 PDF、对话 PDF 各自独立，互不干扰。
- **多服务商可配置**：DeepSeek（默认）/ OpenAI / Kimi / 智谱 / 自定义，前端设置、后端动态读取。
- **不预置 Key**：每个用户填自己的 Key，只存各自浏览器，服务器只转发不存储。
- **自动保存**：文章、笔记、对话存本地，刷新不丢。
- **防冷启动**：`/ping` 端点，配定时任务保活。

## 本地运行

```bash
pip install -r requirements.txt
python app.py          # 打开 http://localhost:8080
```

## 部署

```bash
gunicorn app:app --bind 0.0.0.0:$PORT --workers 1 --threads 8 --timeout 300
```

流式是长连接，`--timeout` 要大；`--threads` 让 512MB 小机器也能并发。

## 防冷启动

用 cron-job.org / UptimeRobot 每 5 分钟访问一次 `https://你的域名/ping`，避免免费主机休眠。

## 导出 PDF 小贴士

浏览器打印对话框里勾选「背景图形 / Background graphics」，红蓝配色和底纹才会完整保留。
