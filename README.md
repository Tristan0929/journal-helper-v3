# 外刊阅读助手 v3

把外刊文章变成个人学习笔记，并练读、写、听、说的小工具。粘贴文章即可，所有数据只存在用户自己的浏览器里，服务器只做转发、不存储任何内容或密钥。

## 功能一览

### 笔记（六板块，流式生成）
- **自动整理版面**：直接粘贴分栏、断行、乱码的原文，AI 先把版面理顺再做笔记，无需手动排版。
- 六个板块：文章展示 → 词汇词组精析（脚注）→ 长难句语法精讲 → 句子赏析 → 写作手法 → 背诵速查表。
- **词汇宽收、句子精选**：词汇尽量多收地道搭配/习语/进阶词；精彩句子按修辞、句法、警句等标准精选。
- 配色克制：<span style="color:#c0392b">红色加粗</span>=重点词汇、*蓝色斜体*=精彩句子、灰色加粗=结构标签（导出 PDF 保留）。
- 脚注 `📖 联动 / 💡 仿写` 自动分行显示。

### 标签页功能
- **🎴 闪卡**：简易 Anki——翻面后用「不认识 / 模糊 / 认识」记熟练度、可「只复习不熟的」；另有 🔊 朗读、🎤 跟读、✍️ 拼写自测；导出 Anki CSV。
- **📇 词汇表**：单独查看背诵速查表，导出 PDF / CSV。
- **✍️ 练习题**：左文右题；按钮点了才生成 5 道阅读理解选择题，即时判分；可朗读全文。
- **🌍 深度思考**：按需生成一段全英文批判性分析；下方可用英文写评论，AI **先润色你的表达再和你讨论**（练写作）。原文与讨论各自可导 PDF。

### 交互与体验
- **选中提问**：笔记里选中词句 → 冒出「💬 问 AI」，点了才带上下文打开侧边栏，可自由提问。
- **停止重生成**：流式生成中可随时中断，省 token。
- **多语言**：默认自动识别原文语种（英/日/德/韩/西等），讲解用中文、深度思考用英文；设置里可手动指定语种兜底。
- **本地自动保存**：文章、笔记、问答、深度思考讨论、闪卡熟练度、练习题全部存浏览器，刷新/重开都在，直到点「🗑️ 清空重来」进入下一篇。
- **导出**：Markdown 一律「复制」，PDF 各板块单独下载。
- **📖 使用教程**：右上角按钮，随时查看详细教程。

### 服务与配置
- **多服务商可配置**：DeepSeek（默认）/ OpenAI / Kimi / 智谱 / 自定义，服务商、base_url、模型、Key 全部前端可配，后端动态读取。
- **不预置任何 Key**：每个用户填自己的 Key，只存各自浏览器。
- **防冷启动**：`/ping` 端点，配定时任务保活。

## 接口

| 路由 | 说明 |
|------|------|
| `POST /analyze` | 生成六板块笔记（流式 SSE） |
| `POST /chat` | 侧边栏 AI 导师（流式） |
| `POST /exercises` | 生成阅读理解题（JSON） |
| `POST /deepdive` | 生成全英文深度思考（流式） |
| `POST /discuss` | 深度思考的讨论 + 写作润色（流式） |
| `GET /ping` | 保活端点 |

## 本地运行

```bash
pip install -r requirements.txt
python app.py          # 打开 http://localhost:8080
```

## 部署（生产）

```bash
gunicorn app:app --bind 0.0.0.0:$PORT --workers 1 --threads 8 --timeout 300
```

流式是长连接，`--timeout` 要大；`--threads` 让单 worker 也能并发，适合 512MB 小机器。

## 防冷启动

免费主机没人访问会休眠。用 cron-job.org / UptimeRobot 每 5 分钟访问一次 `https://你的域名/ping` 即可保持常醒。

## 小贴士

- 闪卡 / 词汇表依赖「背诵速查表」，等笔记整份生成完再切过去。
- 导出 PDF 时在打印对话框勾选「背景图形 / Background graphics」，配色和底纹才完整保留。
- 🎤 跟读用浏览器语音识别，建议用 Chrome；不支持的浏览器会自动隐藏该按钮。
- 深度思考的讨论建议用更强的模型（如 `deepseek-v4-pro`），润色和讨论质量更好。
